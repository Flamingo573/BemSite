##Запуск сборки

**Если установлен глобально Gulp4**
```
gulp
```

**Если нет**
```
npm run gulp
```

##Итог
Если лень клонировать репозиторий, то с результатом сборки можно ознакомиться [тут](http://flamingo-web.ru/site/BemSite/).
